<p><i>XryptoMail</i> is an email client for Android supporting the following features:</p><ul><li>Push IMAP IDLE and POP support</li><li>WebDAV support</li><li>Multiple mail accounts support</li><li>User OAUTH2 authentication support</li><li>Message list sorting by date, arrival, sender, read state etc</li><li>Message list with different backgrounds for read state, encryption and stealth etc</li><li>Message refiling and flagging</li><li>Message view gesture support with Pinch-to-Zoom</li><li>Message view/forward with html content support</li><li>Message send signature support</li><li>Multiple folders support e.g. Gmail</li><li>Android runtime app permission request</li><li>Dual Theme Support</li><li>Multiple languages UI support</li><li>GPG &amp; PGP/MIME support using OpenKeychain</li><li>Autocrypt Level 1 basic features support</li><li>Stealth message with auto deletion on read timeout</li></ul>
